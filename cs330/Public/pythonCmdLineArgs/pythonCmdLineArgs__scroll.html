1 Using Python

Python code can be executed directly by a Python interpreter. The code for this course can be run with Python 3.11 or newer.

1.1 Running a Python Program

You can run any provided Python code with…

python3.11 {main_python_file}.py

1.2 Running Tests

You can run any included test code with…

python3.11 -m pytest tests -v -ra

You may need to run…

python3.11 -m pip install pytest

if you do not have pytest installed in your Python environment

2 Command Line Arguments

Most (almost all) examples will make use of command line arguments. Take a look at…

import sys


def main():

    for idx, arg in enumerate(sys.argv):
        print(f"{idx:>3}: {arg:<}")


if __name__ == "__main__":
    main()

demo.py is a simple program. All it does is echo any entered command line arguments.

The first argument (i.e., argv[0]) is a special case. It is the program name (demo.py in this case), and it is supplied automatically.

2.1 Playing with Command Line Arguments

Save a copy of everything in the Review-00-Foundational-Concepts/Command-Line-Args-Python directory …then open a Linux terminal (e.g., SSH into a CS Linux server or open a local Linux terminal).

  1. Run

    python3.11 demo.py

    You should see…

      0: demo.py

  2. Run

    python3.11 demo.py 1 2 77 The letter Q "Hello, and..."

    You should see…

      0: demo.py
  1: 1
  2: 2
  3: 77
  4: The
  5: letter
  6: Q
  7: Hello, and...

Do not forget… The double quotes are required for any argument that contains spaces.

2.2 Using Command Line Arguments

Most programs will not simply echo command line arguments. Many programs will use command line arguments to specify input files. Consider the following excerpt…

    if len(sys.argv) < 2:
        print("No input file provided.")
        print(f"Usage: {sys.argv[0]} input_file")

        sys.exit(1)

    with open(sys.argv[1], "r") as shapes_file:
        pass

The first conditional block checks that the required number of arguments were supplied by the user.

    if len(sys.argv) < 2:

In this excerpt, only one argument is required… an input file name. Note that the first true argument (sys.argv[1]) is the input file name.
