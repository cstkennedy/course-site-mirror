2 Accessors vs Mutators

You should recall from your previous coursework that not all accessors are getters and not all mutators are setters. The accessor and mutator rows deal with getters and setters, respectively.

That is a bit more manageable… even with the sneakily added Python column. Take note of how similar Python’s Class Checklist is to the Java Class Checklist.

5 Rote Memorization is Bad

The biggest mistake one can make is to say… I must include all of these in my class. This is a checklist… every item must be considered based on the nature of the class to be implemented.
