1 Using Cargo

All Rust code is accompanied by a Cargo.toml configuration file.

1.1 Compiling and Running a Rust Program

In most cases… you can run any provided Rust code with…

cargo build; cargo run

or

cargo run

The latter command executes, compiles, and then executes the Rust program. This allows us to use fewer keystrokes (which are expensive).

1.2 Compiling and Running Tests

You can compile and run any included test code with…

cargo test

2 Command Line Arguments

Most (almost all) examples will make use of command line arguments. Take a look at…

use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();

    for (idx, arg) in args.iter().enumerate() {
        println!("{:>2}: {}", idx, arg);
    }
}

demo.rs is a simple program. All it does is echo any entered command line arguments.

2.1 Playing with Command Line Arguments

Save a copy of everything in the Review-00-Foundational-Concepts/Command-Line-Args-Rust directory

…then open a Linux terminal (e.g., SSH into a CS Linux server or open a local Linux terminal). Run ./gradlew to compile Demo.rust.

  1. Run

    cargo run

    You should see…

     0: target/debug/Command-Line-Args-Rust

  2. Run 

    cargo run -- 1 2 77 The letter Q "Hello, and..."

    You should see…

     0: target/debug/Command-Line-Args-Rust
 1: 1
 2: 2
 3: 77
 4: The
 5: letter
 6: Q
 7: Hello, and...

Do not forget… The double quotes are required for any argument that contains spaces.

2.2 Using Command Line Arguments

Most programs will not simply echo command line arguments. Many programs will use command line arguments to specify input files. Consider the following excerpt…

    let argv: Vec<String> = env::args().collect();                                                                          
                                                                                                                            
    if argv.len() < 2 {                                                                                                     
        println!("Usage: {} file_name", argv[0]);                                                                           
        std::process::exit(1);                                                                                              
    }

In this excerpt, only one argument is required… an input file name. Note that the first true argument (argv[1]) is the input file name.
