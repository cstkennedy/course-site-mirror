1 The Problem

Write a program to compute an approximate value for the derivative of $f(x)$ using the finite difference formula

$$ f^\prime(x) \approx \frac{f(x+h)-f(x)}{h} $$

2 Input

There is no input to this program. All values and functions will be hardcoded.

We will use

  • $f(x) = \sin(x)$
  • $x = 1$

3 Output

Your output should be similar to…

|  h   |       x       | Approx. f'(x) |  Known f'(x)  |  Abs. Error   |
|:----:|--------------:|--------------:|--------------:|--------------:|
|2^-01 |    1.00000000 |    0.31204800 |    0.54030231 |    0.22825430 |
|2^-02 |    1.00000000 |    0.43005454 |    0.54030231 |    0.11024777 |
|2^-03 |    1.00000000 |    0.48637287 |    0.54030231 |    0.05392943 |
|2^-04 |    1.00000000 |    0.51366321 |    0.54030231 |    0.02663910 |

However, unlike this abbreviated example you must complete up to (2^{-30}).

Note… that your output format must be the same. That actual values that you compute may differ due to finite precision.

4 Your Tasks

For $h=\frac{1}{2}, \frac{1}{4}, \frac{1}{8}, \dots \frac{1}{2^{30}}$

  1. Compute an approximate value (Approx. f'(x)) for $f^{\prime}(x)$ using…

    $$ f^\prime(x) \approx \frac{sin(1+h)-sin(1)}{h} $$

  2. Compute the known value for $f^{\prime}(x)$ (Known f'(x)) using cos(1). Note… that you can compute this value once and reuse it.

  3. Compute the absolute error (Abs. Error) between the values computed in the previous two (2) steps.

Generate a summary of your results.

  1. Approximate eps using the Cleve Moler Algorithm.

  2. Compute $\sqrt{eps}$.

  3. Plot h (x-axis) vs Abs. Error (y-axis).

    1. Set both axes (x-axis and y-axis) to logarithmic scales

    2. You may generate the graph as part of your program or using a separate program (e.g., Excel or Desmos).

  4. Examine the graph and answer the following…

    1. What is the minimum value for the magnitude of the error?

    2. How does it compare to $\sqrt{eps}$?

  5. Include your findings in your zip file as one of (or a combination of) the following…

    1. Output generated by running your program.

    2. An Excel file with the graph.

    3. An image file (e.g., PNG or SVG) containing your graph.

    4. Part of your README.md file.
